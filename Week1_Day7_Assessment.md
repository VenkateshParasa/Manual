- Review previous day's content before starting new
- Practice exercises immediately after learning

### Learning Strategy
- **Understand, Don't Memorize:** Focus on concepts, not rote learning
- **Practice Actively:** Write test cases, create documents
- **Ask Questions:** Clarify doubts immediately
- **Teach Others:** Explain concepts to solidify understanding
- **Real-World Application:** Think how to apply in actual projects

- **Time Management:** 1.2 minutes per question
- **Don't Rush:** Read questions carefully

3. Not understanding when to use which test design technique
4. Incomplete test case documentation

6. Not calculating metrics correctly
