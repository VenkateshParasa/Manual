   a) 10%

   d) 100%

**7. Which Agile Testing Quadrant focuses on exploratory testing?**
   a) Q1
   b) Q2
   c) Q3
   d) Q4

**8. What does DevOps combine?**
   a) Development + Operations
   b) Development + Quality Assurance
   c) Deployment + Operations
   d) Design + Operations

**9. In Canary release, what percentage of users typically get the new version first?**
   a) 1-5%
   b) 25-50%
   c) 50-75%
   d) 100%

**10. What is the main benefit of the whole team approach in Agile?**
   a) Faster development
   b) Shared quality ownership
   c) Less documentation
   d) No need for QA

**11. Which CI/CD metric measures time from commit to production?**
   a) Build duration
   b) Deployment frequency
   c) Lead time
   d) MTTR

**12. What does MTTR stand for?**
   a) Mean Time To Release
   b) Mean Time To Recovery
   c) Maximum Time To Repair
   d) Minimum Testing Time Required

**13. In Agile Testing Quadrants, which quadrant is 100% automated?**
   a) Q1
   b) Q2
   c) Q3
   d) None are 100% automated

**14. What is the target change failure rate in DevOps?**
   a) <5%
   b) <15%
   c) <25%
   d) <50%

**15. Which testing type belongs in Q4 (Technology-facing, Critique)?**
   a) Unit tests
   b) Functional tests
   c) Performance tests
   d) Exploratory tests
